# 🌐 Paróquia Nossa Senhora da Cabeça

Site informativo desenvolvido para a Paróquia Nossa Senhora da Cabeça, com o objetivo de facilitar o acesso às principais informações da comunidade.

---

## 📍 Funcionalidades

- 📅 Programação semanal das missas
- 👨‍⛪ Informações do pároco
- 📢 Seção de eventos
- 📞 Contato da paróquia
- 📖 Versículos e salmos dinâmicos
- 📱 Layout responsivo (funciona em celular e computador)

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

---

## 🌐 Acesso ao site

👉 https://joalyssonsouza.github.io/site-paroquia/

## 🎯 Objetivo

Projeto desenvolvido como atividade de extensão acadêmica, com foco em contribuir com a comunidade através da tecnologia.

## 📌 Observações
